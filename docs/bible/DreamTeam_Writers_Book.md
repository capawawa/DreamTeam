How to Use This Book
- This is a prescriptive manual. Each chapter sets: purpose → inputs → rules/checklists → deliverables → continuity hooks → approvals.
- Follow page targets. Favor bullets over prose.
- Cite sources with clickable file references; if line is unknown, include “line TBD”.

Glossary (minimal)
- Ambiguity discipline: Rule that any sinister cue must have a normal POV counter within 1–2 pages. Sources: ["Canon_Rules.md"](docs/canon/Canon_Rules.md:11-14), ["DreamTeam_Research_Inventory.md"](docs/research/DreamTeam_Research_Inventory.md:92-99)

- Vendor exception: Dialogue with vendors/cashiers renders normally even when characters are high. Sources: ["Canon_Rules.md"](docs/canon/Canon_Rules.md:19-20)
- Corral: In-story tactic to contain drunken crowds within biergarten boundaries. Sources: ["Festival_Missions.md"](docs/missions/Festival_Missions.md:7-9)

### 2.1 Character 2-Page Brief Template

Page 1
- Identity: Name; role; one-line core descriptor.
- Desire/Need: External goal; internal need.
- Flaw/Vulnerability: What derails them under festival stress.
- Tactics: How they try to win scenes.
- Relationships: 2–3 sentence network.
- Visual Tells: Clothing, posture, props.

Page 2
- Arc Beats (I1–I5):
  - I1 → I2 → I3 → I4 → I5 (one line each).
- Do/Don’t List:
  - Do: stylistic and behavioral greenlights consistent with canon.
  - Don’t: content/taboos, tone breakers.
- Sample Beats: 3 beat ideas with POV notes and ambiguity counters.
- Citations: files/lines confirming traits.

Completed Example (Greg)

- Flaw: Susceptible to distraction by “evil.” ["Character_Bible.md"](docs/characters/Character_Bible.md:13-16)

- Do: Pair insight beats with Matt POV counters within 1–2 pages. ["Canon_Rules.md"](docs/canon/Canon_Rules.md:21-22)
