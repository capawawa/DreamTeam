## Comic Script Format — Guide

- Page heading: `PAGE N (Panels)`
- Panel heading: `Panel X:` concise action, key visual focus.
- Dialogue: `CHARACTER: text` (keep balloons short).
- Captions: `CAPTION (Name): text`
- SFX: `SFX: text`
- Art notes: camera, mood, motion cues when essential.

### POV Convention (This Project)
- Dreamteam POV: allow trippy overlays and stylized cues in panel notes.
- Matt POV: explicitly note “Matt POV” for clean, normal rendering.
