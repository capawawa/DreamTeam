## Production Plan — Milestones & Roles

- Issues: 5 (25–32 story pages each) + 1–3 pages of extras per issue.
- Roles: writing, pencils, inks, colors, letters, editing.
- Milestones (per issue):
  - Outline lock → Script v1 → Thumbs → Pencils → Inks → Colors → Letters → Proof → Final.
- Timeline target: 6–8 weeks per issue with overlap.
- Assets: maintain `/docs`, `/outline`, `/scripts`, `/production`, version control.
